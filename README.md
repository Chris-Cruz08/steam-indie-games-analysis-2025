# The Economics of Indie Games on Steam (2025)

Analysis of what makes an indie game successful on Steam using completely open data from SteamSpy[](https://steamspy.com/api.php).

## Features
- Downloads live data for the top 1000 most-owned games
- Calculates "Playtime per Dollar" metric
- Separates Indie vs AAA titles
- Top 20 best-value indie games of all time
- Beautiful charts saved in /output

## How to run
```bash
pip install -r requirements.txt
python main.py
